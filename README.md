# Gallog Cleaner

크로미움 기반 브라우저(크롬, 엣지, 웨일 등)에서 돌아가는 디시인사이드 갤로그 클리너입니다.
디시인사이드 갤로그 화면에서 글, 댓글 삭제를 도와줍니다.

## 설치

1. Chrome 주소창에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 누릅니다.
4. 이 프로젝트 폴더를 선택합니다.
5. 확장 프로그램을 새로 로드한 뒤 갤로그 페이지를 새로고침합니다.

## 사용 방법

1. `https://gallog.dcinside.com/...` 갤로그 페이지로 이동합니다.
2. 왼쪽 메뉴에서 정리할 섹션을 직접 엽니다. 예: 게시글, 댓글, 스크랩, 방명록
3. 오른쪽 아래 `Gallog Cleaner` 패널에서 `삭제 후보 미리보기`를 누릅니다.
4. 처음 사용하는 환경에서는 `이번 실행`을 `1`로 낮춰 정상 동작을 확인합니다.
5. 문제가 없으면 삭제 개수와 삭제 간격을 원하는 값으로 조정합니다.

## 갤러리 필터

`갤러리 필터`는 현재 페이지의 갤러리 목록을 읽어 실제 갤러리 이름으로 보여줍니다.

- 갤러리를 선택하면 `선택된 갤러리`에 칩으로 추가됩니다.
- 칩을 누르면 선택에서 제거됩니다.
- `선택한 갤러리만 삭제`는 선택한 갤러리 항목만 후보로 삼습니다.
- `선택한 갤러리는 제외`는 선택한 갤러리 항목을 건너뜁니다.
- 필터가 켜져 있는데 항목의 갤러리를 확인하지 못하면 안전하게 삭제 후보에서 제외합니다.

## 기본값

처음 설치했을 때의 기본 설정은 다음과 같습니다.

| 항목 | 기본값 |
| --- | --- |
| 이번 실행 | `100` |
| 삭제 간격 | `1000ms` |
| 다음 페이지 | 꺼짐 |
| 갤러리 필터 방식 | 선택한 갤러리만 삭제 |
| 선택된 갤러리 | 없음 |
| 한 번에 설정 가능한 최대 삭제 시도 개수 | `100000` |
| 최소 삭제 간격 | `500ms` |
| 최대 삭제 간격 | `10000ms` |

사용자가 바꾼 설정은 Chrome 확장의 `chrome.storage.local`에 저장됩니다. 저장되는 주요 값은 삭제 간격, 이번 실행 개수, 다음 페이지 옵션, 갤러리 필터 방식, 선택된 갤러리 목록입니다. 이미 설치해서 사용한 적이 있다면 저장된 기존 설정이 새 기본값보다 우선 적용됩니다.

실행 중 작업 상태도 같은 저장소에 `gcActiveJob`으로 임시 저장됩니다. 여기에는 남은 개수, 시도 개수, 실패 개수, 현재 섹션 범위, 리캡챠 일시중지 여부가 들어갑니다. 이 작업 상태는 완료하거나 `중지/취소`를 누르면 삭제되며, 오래된 작업은 30분 뒤 자동 폐기됩니다.

## 안전장치

- 삭제는 화면에 실제로 보이는 삭제 버튼만 대상으로 합니다.
- 실행 중에만 삭제 확인창을 자동 승인합니다.
- `중지/취소` 버튼은 실행 중인 작업과 저장된 이전 실행을 취소합니다.
- 삭제 후 페이지가 새로고침되면 남은 작업을 같은 갤로그 섹션에서 이어서 진행합니다.
- 다음 페이지 이동은 현재 페이지 번호의 다음 숫자 링크를 먼저 사용하고, 10페이지 묶음 끝에서만 다음 묶음 버튼을 사용합니다.
- 리캡챠나 자동화 확인 화면이 감지되면 작업을 일시중지하고 시스템 알림을 보냅니다.
- 리캡챠를 직접 확인한 뒤 `계속` 버튼으로 남은 작업을 이어갈 수 있습니다.
- 삭제 중 `글 번호가 올바르지 않습니다.` 사이트 알림이 뜨면 자동 확인하고 실패 카운트에 기록합니다.

삭제는 되돌릴 수 없습니다. 반드시 `삭제 후보 미리보기`와 소량 테스트를 먼저 사용하세요.
